## A multilingual entry point for two core modules

The CPcashLabs DAO site is organized around two clear modules: **Roadmap** and **Recruiting**.

This structure makes it easier to explain where the ecosystem is heading and who we want to build it with.

## Module 1: [Roadmap](./roadmap/)

The roadmap module collects the strategic direction of CPcashLabs DAO:

- [Ecosystem Overview](./roadmap/ecosystem/)
- [Vision](./roadmap/vision/)
- [Roadmap Whitepaper](./roadmap/whitepaper/)

This module is meant to guide long-term DAO direction, coordination priorities, and ecosystem expansion.

## Module 2: [Recruiting](./recruiting/)

The recruiting module is the current execution layer for community building:

- [Recruiting Overview](./recruiting/)
- [Community Volunteer](./recruiting/community-volunteer/)
- [Web3 Community Business Expert](./recruiting/web3-community-business-expert/)
- [How to Join](./recruiting/join-us/)

This module is where we turn values into contributors, relationships, and working capacity.

## Suggested reading path

- If you want to understand the direction first, start with the [Roadmap module](./roadmap/)
- If you already want to participate, go directly to the [Recruiting module](./recruiting/)

## What CPcashLabs DAO is building

Through CPcashLabs DAO, we want to build a privacy-first Web3 ecosystem where contributors can organize, communities can coordinate, and personal financial freedom can be protected in a decentralized way.

Your money belongs to you. Your life belongs to you. No one should be able to casually observe, control, or take either away.
